## 🚀 使用方法

### 立即测试新数据源
```bash
# 专门测试厂商爬虫
python3 test_manufacturer_crawler.py

# 完整测试（包含新数据源）
python3 test_crawler.py

# 立即开始全面检索
python3 run_crawler.py
```

### 配置选项
在 `config.py` 中可以调整：
- `PHONE_MANUFACTURER_SOURCES`: 手机厂商网站列表
- `TECH_COMPANY_SOURCES`: 技术公司网站列表
- `REQUEST_DELAY`: 请求间隔（避免过于频繁）
- `MAX_CONCURRENT_REQUESTS`: 最大并发数
